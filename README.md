# Swiggy-Sales-Performance-Dashboard-Excel
📊 Swiggy Sales Dashboard built in Excel to analyze ₹53M+ sales data. Features interactive KPIs, sales trends (daily, weekly, monthly), category insights, and state-wise performance. Helps identify customer behavior, top cities, and key business insights using Pivot Tables and charts.

📌 Project Overview

This project presents a comprehensive Sales Analysis Dashboard for Swiggy, developed using Microsoft Excel. The dashboard transforms raw food delivery data into meaningful insights, enabling better understanding of sales trends, customer behavior, and regional performance.

It is designed to support data-driven decision-making through interactive visuals and dynamic filtering.

🎯 Objectives

The main goals of this project are:

To analyze overall sales performance
To track order trends across different time periods
To identify top-performing cities and states
To understand customer preferences (Veg vs Non-Veg)
To evaluate key performance indicators (KPIs)
To build an interactive and user-friendly dashboard


📊 Key Performance Indicators (KPIs)

The dashboard highlights the following key metrics:

Total Sales: ₹53.01 Million
Average Rating: 4.34 ⭐
Average Order Value: ₹268.51
Total Orders: 197.4K
Total Ratings Count: 5.6 Million

These KPIs provide a quick snapshot of business performance.


📈 Dashboard Features

🔹 Sales Trend Analysis

Monthly sales trend to identify seasonal patterns
Daily sales trend to track weekday performance
Weekly trend to highlight peak sales periods

🔹 Category Analysis

Comparison between Veg and Non-Veg sales
Helps understand customer ordering preferences

🔹 Geographic Insights

State-wise sales distribution using map visualization
Identifies high and low-performing regions

🔹 Top Performing Cities

Displays top 5 cities based on total sales
Helps identify key revenue-generating locations

🔹 Interactive Filters

The dashboard includes dynamic slicers for:

Month selection
Food category
Restaurant

These filters allow users to explore customized insights.

💡 Key Insights

Non-Veg category contributes ~64% of total sales, showing higher demand.
Bengaluru is the top-performing city, generating the highest revenue.
Sales show consistent growth with peak activity during weekends.
High average rating (4.3+) indicates strong customer satisfaction.
